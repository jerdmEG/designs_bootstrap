Usage

You can use this icon in React, Ruby, or Jekyll. Here are code examples for each:

React

<CheckCircleFillIcon size={12} />

ViewComponent

<%= render(Primer::Beta::Octicon.new(:"check-circle-fill", size: :xsmall)) %>

Jekyll

{% octicon check-circle-fill height:12 %}
